# Develop a Program for a Gym
## Features:
• Sign in page for customers, employee, and admins.
## Customers
• Can create an account with a username, password, full name, and address.

• Can sign up for a gym membership (they can choose to pay monthly or yearly).

Note: you do not need to implement real payment processing into your program.

• Can view and book fitness classes from a schedule (e.g., yoga, palates, etc…).

• Can book one-on-one appointments with trainers.

• Can view their fitness class and trainer appointment history.

• Can update their name, address, and contact information after signing in.

## Employees – an employee can be a receptionist, fitness coach, or a trainer.

• (If receptionist) Can view a list of all members, including their memberships, booked
classes, and appointments with trainers.

• (If receptionist) Can cancel a customer’s gym membership.

• (If fitness coach) Can create fitness classes (include class type and time of class).

• (If fitness coach) Can cancel fitness classes when necessary.

• (If trainer) Can accept or decline training appointments made with them by customers.

• (If trainer) Can cancel training appointments when necessary.
Admin Account

• Can create employee accounts with a username, password, full name, and employee
type (fitness coach, trainer, or receptionist).

## Extra feature (optional): try to set it up so when an employee signs in for the first time, using the password set by the admin, the employee is asked to enter a new password.

• Can modify or delete customer and employee accounts.

• Can delete class schedules, memberships, and trainer appointments.

• Can reset employee or customer passwords.
